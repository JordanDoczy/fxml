Lesson 5. The Function Keyword.

In this example you will learn about how to use the function keyword.

The function keyword is used to call functions that return a value.

As we've seen in Lesson 2. The Basics. calling a method is quite simple. However, in that example the method called did not return a value. But what if you need to retrieve the value of a function and assign it to a variable? In that case you must use the function keyword.
